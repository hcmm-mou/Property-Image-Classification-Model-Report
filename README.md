# Property Image Classification Model Report & Research

## Project Overview 
This project completed training notebook and evaluated the classification model to automate the classification of property images for Real Estate Agency X (case study). The agency processes approximately 50,000 new properties monthly, which required manual categorization and processing of photos into categories such as living room, bedroom, kitchen, bathroom, and exterior, a process that previously consumed ~**16,650 hours** of manual labour per month for workers.
